# AtividadesCG

Repositorio para as entregas da Atividade Academica de Computacao Grafica.

## Projeto atual

- Hello3D (C++ com OpenGL moderna via GLFW)

## Requisitos (macOS)

- CMake 3.20+
- Xcode Command Line Tools

## Build e execucao

```bash
cmake -S . -B build
cmake --build build
./build/Hello3D
```
